# homebrew-tele

Homebrew tap for [tele](https://github.com/sorokin-vladimir/tele) — keyboard-driven TUI Telegram client.

## Install

```sh
brew tap sorokin-vladimir/tele
brew install tele
```

## Upgrade

```sh
brew upgrade tele
```

## Uninstall

```sh
brew uninstall tele
brew untap sorokin-vladimir/tele
```
